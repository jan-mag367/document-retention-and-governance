# Access controls (draft)

## Roles
- Admin: technical management in sandbox
- Owners: manage sites/Teams and permissions
- Editors: create and edit documents
- Viewers: read-only access

## Principles
- Least privilege by default
- Restrict governance/finance folders
- Avoid sharing to personal emails
- Time-box temporary access

## Reviews
- Monthly check of external sharing links
- Quarterly permissions review (owners/editors/viewers)
