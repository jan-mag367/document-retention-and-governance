# Naming conventions (draft)

## Goals
- Easy sorting and searching
- Clear version history
- Consistent file names across folders

## Recommended format
YYYY-MM-DD_topic_area_vX

Examples:
- 2026-02-02_minutes_branch-meeting_v1
- 2026-02-02_policy_data-handling_v2
- 2026-02-02_campaign_plan_v1

## Rules
- Date first (ISO format)
- Avoid special characters
- Keep names short and descriptive
- Use version numbers only when needed
